Comecei na carreira de desenvolvimento através de um curso técnico em informática, onde aprendi conceitos hardware, software e design. Me aperfeiçoei (e continuo me aperfeiçoando) na parte de desenvovimento de softwares e design, principalmente web.

Hoje estou aprendendo sobre o React, uma das principais bibliotecas JavaScript para criação de frontend para a web. Já no backend estou desenvolvendo um framework PHP para criação de APIs REST, futuramente pretendo ampliar meus conhecimentos em Python, tanto para criação de banckend web como inteligência artificial.

Além de tecnologias e linguagens, sempre estou estudando os conceitos por traz delas. Acredito que o código deve ser o mais simples possível e que só deve adicionar complexidade e/ou dependências quando essas oferecem uma real vantagem. Gosto de desenvolver baseado em protótipos, criando versões básicas funcionais e implementando mais funcionalidades e segurança conforme for aprovado.

Minhas principais habilidades são proatividade, autodidatismo e comunicação, tenho ideias para diversos projetos, que vou executando no meu tempo livre. Programação não é só uma profissão, mas um lazer.
